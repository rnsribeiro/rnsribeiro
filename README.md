## Sobre mim

Desenvolvedor Full Stack com atuação em arquitetura de sistemas, desenvolvimento de software, code review e deploy em homologação e produção. Minha trajetória reúne experiência com aplicações web, APIs, infraestrutura, automação, inteligência artificial aplicada, Internet das Coisas e sistemas embarcados.

Também tenho interesse contínuo em construir soluções práticas, com foco em desempenho, manutenção e evolução técnica constante.
